Sincronizare între dispozitive

Datele tale se salvează criptate într-un gist privat GitHub. Le poți încărca pe orice dispozitiv cu același token și aceeași parolă. Nimeni — nici GitHub — nu le poate citi fără parola ta.

Linkul conține tokenul și parola — trimite-l DOAR familiei tale, prin mesaj privat. Cine îl are vede și modifică datele.

Cum obțin tokenul GitHub?
  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic).
  2. La „Select scopes" bifează DOAR căsuța gist. Nimic altceva.
  3. Generează, copiază tokenul (ghp_…) și lipește-l mai sus. Pune și o parolă de criptare pe care o ții minte.

Tokenul stă doar local, pe dispozitivul tău. Cu scope „gist" nu poate atinge codul tău, doar gist-urile.

Gemini AI (cheie & model)
  1. Intră pe aistudio.google.com și loghează-te cu contul tău Google.
  2. Apasă „Get API key” (stânga sus) → „Create API key”.
  3. Copiază cheia (începe cu AIza…) și lipește-o mai jos.

Notă: abonamentul Gemini Pro din aplicația Gemini e separat de API. Cheia din AI Studio are un nivel gratuit — alege un model „flash".

Dacă modelul atinge limita zilnică (429), aplicația trece automat pe alt model gratuit. Cheia se salvează doar local. Asistentul din chat folosește mereu un model rapid (flash-lite) ca să răspundă cât mai repede.

Notificări push (telefon & desktop)

Primești notificările FamLink direct în browser — apar instant pe telefon și desktop, chiar și când FamLink e închis. Rulează gratuit prin Firebase Cloud Messaging (Google). Configurezi o singură dată; setarea ajunge automat și la ceilalți membri prin sincronizare.

Ghid de activare (o singură dată, de pe computer)
  1. Urmează pașii din fișierul PUSH-SETUP.md (creezi un proiect Firebase, completezi firebase-config.js și dai deploy la backend cu firebase deploy).
  2. La final primești un URL al funcției (se termină în …/api sau …run.app). Lipește-l mai jos.
  3. Pe telefon: deschide FamLink, apasă „Activează notificările" și acceptă. Apoi apasă Testează.

Pe iPhone, notificările push în web funcționează doar dacă adaugi FamLink pe ecranul principal (Share → „Add to Home Screen") și deschizi aplicația de acolo.

Mai primești emailuri de la FamLink? Ele NU vin din aplicație, ci din vechiul script rămas în contul tău Google — el are un ceas propriu care rulează la 1 minut și trimite ce are în coadă. Aplicația îi golește coada automat, dar ca să-l oprești definitiv șterge-i trigger-ul (o singură dată):

  1. Apasă butonul de mai jos (loghează-te cu contul tău Google dacă ți-l cere).
  2. Găsește rândul cu proiectul FamLink („FamLink notificări") → cele 3 puncte din dreapta → Delete trigger.
  3. Opțional, curat de tot: script.google.com → proiectul FamLink → Deploy → Manage deployments → Archive.
