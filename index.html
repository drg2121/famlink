Sincronizare între dispozitive

Datele tale se salvează criptate într-un gist privat GitHub. Le poți încărca pe orice dispozitiv cu același token și aceeași parolă. Nimeni — nici GitHub — nu le poate citi fără parola ta.

Cum obțin tokenul GitHub?
  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic).
  2. La „Select scopes" bifează DOAR căsuța gist. Nimic altceva.
  3. Generează, copiază tokenul (ghp_…) și lipește-l mai sus. Pune și o parolă de criptare pe care o ții minte.

Tokenul stă doar local, pe dispozitivul tău. Cu scope „gist" nu poate atinge codul tău, doar gist-urile.

Gemini AI (cheie & model)
  1. Intră pe aistudio.google.com și loghează-te cu contul tău Google.
  2. Apasă „Get API key” (stânga sus) → „Create API key”.
  3. Copiază cheia (începe cu AIza…) și lipește-o mai jos.

Notă: abonamentul Gemini Pro din aplicația Gemini e separat de API. Cheia din AI Studio are un nivel gratuit — alege un model „flash".

Dacă modelul atinge limita zilnică (429), aplicația trece automat pe alt model gratuit. Cheia se salvează doar local.
